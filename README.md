# Lollipop

> Under heavy development

## Description
A web based MP3 player that plays differnt files on left and right stereo channels. For use when two peoples want to listen to different songs with same device and headphone.

## Authors
Yoganathan P

## Build Setup

```bash
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server
$ npm run build
$ npm run start

# generate static project
$ npm run generate
```

For detailed explanation on how things work, check out [Nuxt.js docs](https://nuxtjs.org).
